NEWS (JANUARY 10, 2025)

DOOM: The Gallery Experience v1.2.1

An update to DOOM: The Gallery Experience has been released. The major changes include:

  • Saving and loading has been overhauled and optimized
    • Saves have a new format
    • Standalone saves can be found in "(user)/AppData/Local/SCUMDOG/doom_tge"
    • Fullscreen status is now saved for standalone version
    • Loading now happens as soon as the game is launched
  • Improved controller support
    • Left joystick movement now utilizes full analog
    • Menus are now navigable by left joystick
  • Changed gift shop purchase sound
  • Fullscreen with F is now allowed at any point in the game, including the splash screen
  • Refactored and optimized menu code
  • Added SCUM DOG GAMES logo to splash screen and credits
  • Included a readme file in the standalone version
  • Miscellaneous bug fixes and improvements
