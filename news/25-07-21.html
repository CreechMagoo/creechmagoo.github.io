NEWS (JULY 21, 2025)

DOOM: The Gallery Experience v1.3.0

An update to DOOM: The Gallery Experience has been released. The major changes include:

  • Added marquee effect for long artwork titles
    • Updated many artworks to use their full titles
  • Mapped the X button to drink on gamepad
  • The exit switch now animates and plays a sound when used
  • Refactored results screen
    • Will now auto-advance after 30 seconds
    • Larger stats will now tally much faster
  • The game will now wait on the startup screen until save data is loaded, if any exists
  • Fixed cases where the idle timeout would run when it shouldn't
  • Fixed cases of the camera jolting when viewing artworks (thanks Joey)
  • Fixed main menu panorama movement incorrectly scaling with higher framerates
  • Fixed grave markers in Egyptian wing showing swapped preview images
  • Various other bug fixes and improvements
