News (January 6, 2025)

DOOM: The Gallery Experience v1.2.0

An update to DOOM: The Gallery Experience has been released. The major changes include:

  • The game will now save your options and medals
    • This can be cleared by holding the DELETE key on the main menu for five seconds
  • Added rudimentary gamepad support
  • Added a manual written by Newgrounds user ErasmusMagnus to the Windows build
  • Repacked the Windows build
  • Removed the "click to play" screen on the Windows build
  • Fixed the back button in the options menu not showing
  • Miscellaneous bug fixes and improvements
