NEWS (FEBRUARY 3, 2025)

DOOM: The Gallery Experience v1.2.3

An update to DOOM: The Gallery Experience has been released. The major changes include:

  • Added a unique border for windowed mode and nonstandard resolutions
  • The player will now start sleeping halfway through the idle timer
  • Refactored the menu melt effect, slightly improving responsiveness
  • Fixed a bug where gamepads couldn't interact with the results screen
  • Erasing your save file in standalone version now also deletes save folder
  • Updated readme in standalone version with note about streaming programs (#5)
    • If OBS isn't recording the game, try setting Capture Mode to "Windows 10"
