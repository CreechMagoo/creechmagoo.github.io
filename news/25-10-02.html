NEWS (OCTOBER 2, 2025)

DOOM: The Gallery Experience v1.4.0

An update to DOOM: The Gallery Experience has been released. The major changes include:

  • Rebuilt every part of the game from the ground up
    • Moved a lot of hard-defined data to individual files
    • Optimization and response times should be improved across the board
    • This took a lot of time and planning; please report any bugs on the game's issue tracker
  • Added skins!
    • Skins are a new type of unlockable content, similar to medals
    • Newgrounds exclusive: Added Tankman skin for Newgrounds Supporters
  • Updated medals
    • Added 2 new medals
    • Adjusted unlock requirements for several medals
    • Adjusted all medal images
  • Added 'Ingame Timer' option, for speedrunners
  • Added longform time to results screen
  • Updated menus
    • Added Controls menu
    • Shows basic controls that reflect the control scheme you're using
    • Added Unlocks menu
    • Skins, medals, and unlockable options now live here
    • Added fade to menus on the title screen for legibility
    • Added saving indicator
  • Overhauled the level layout
    • Added doors, which require interaction to open
    • Added maps, which show where you are in the gallery and what lies ahead
    • Added and adjusted several textures
    • Edited several sections of the level to be more faithful to E1M1
    • Level geometry that is impossible to view will no longer draw
    • Made the skybox pretty
  • Fixed camera being tilted slightly up
  • Added rudimentary line of sight check to viewing artwork
  • Updated loading screen
    • Overhauled appearance
    • Loading screen now accounts for save and all individual files
    • Added error handling
  • Fixed loads of bugs and inconsistencies
